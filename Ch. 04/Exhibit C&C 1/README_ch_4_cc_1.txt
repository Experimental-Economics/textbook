================================================================================
Chapter 4 - Exhibit C&C 1
Multiple Hypothesis Testing (Karlan and List, 2007)
================================================================================

OVERVIEW
--------
This exhibit demonstrates multiple hypothesis testing corrections using the
mhtexp2 package, based on the methodology from List, Shaikh, and Vayalinkal
(2023). It replicates the analysis from the textbook using the Karlan and
List (2007) charitable giving experiment.

The code runs 10 specifications:
  - Multiple outcomes (with and without controls)
  - Multiple subgroups (with and without controls)
  - Multiple treatments (with and without controls)
  - Pairwise treatment comparisons (with and without controls)
  - Full combination: outcomes + subgroups + treatments (with and without controls)

Each specification produces a LaTeX table and an HTML table in the output/ folder.


FOLDER STRUCTURE
----------------
  code/     Scripts in Python, R, and Stata (any one will produce the results)
  data/     Karlan and List (2007) replication dataset (.dta)
  output/   LaTeX (.tex) and HTML (.html) tables generated by the scripts


PREREQUISITES
-------------
  The mhtexp2 package must be installed for the language you plan to use.
  All three implementations are maintained at:
    https://github.com/vayalinkal/mhtexp2

  Python:  pip install git+https://github.com/elliotjames-paschal/mhtexp2python.git
  R:       Installed automatically when you run the script (via devtools from GitHub)
  Stata:   Download mhtexp2.ado and mhtexp2.sthlp from the repository above
           and place them in your personal ado directory. To find this directory,
           type "sysdir" in Stata (typically ~/ado/personal/ or
           C:\Users\<username>\ado\personal\).


HOW TO RUN
----------

  Python:
    1. Install dependencies (once):
         pip install pandas pyreadstat git+https://github.com/elliotjames-paschal/mhtexp2python.git
    2. Run the script:
         python code/ch4_exh_cc_4.py

  R:
    1. Open code/ch4_exh_cc_4.R in RStudio and run it.
       All required packages (haven, xtable, mhtexp2r) will be installed
       automatically if missing.

  Stata:
    1. Install mhtexp2 (see Prerequisites above).
    2. Set your working directory to the code/ folder:
         cd "path/to/Ch. 4/Exhibit C&C 1/code"
    3. Run the script:
         do ch4_exh_cc_4.do
       The estout package (for esttab) will be installed automatically if missing.

  All three languages produce the same 10 LaTeX tables and 10 HTML tables in output/.


OUTPUT FILES
------------
  LaTeX tables (.tex):
  tab_outcomes.tex          Multiple outcomes
  tab_outcomes_ctrl.tex     Multiple outcomes (with controls)
  tab_subgroups.tex         Multiple subgroups
  tab_subgroups_ctrl.tex    Multiple subgroups (with controls)
  tab_treatments.tex        Multiple treatments
  tab_treatments_ctrl.tex   Multiple treatments (with controls)
  tab_pairwise.tex          Pairwise treatment comparisons
  tab_pairwise_ctrl.tex     Pairwise treatment comparisons (with controls)
  tab_full.tex              Full: outcomes + subgroups + treatments
  tab_full_ctrl.tex         Full: outcomes + subgroups + treatments (with controls)

  HTML tables (.html):
  tab_outcomes.html         Multiple outcomes
  tab_outcomes_ctrl.html    Multiple outcomes (with controls)
  tab_subgroups.html        Multiple subgroups
  tab_subgroups_ctrl.html   Multiple subgroups (with controls)
  tab_treatments.html       Multiple treatments
  tab_treatments_ctrl.html  Multiple treatments (with controls)
  tab_pairwise.html         Pairwise treatment comparisons
  tab_pairwise_ctrl.html    Pairwise treatment comparisons (with controls)

  LaTeX tables use booktabs formatting and include \caption and \label, ready for
  direct inclusion in a LaTeX document via \input{tab_outcomes.tex}.

  HTML tables are self-contained files with inline CSS styling, viewable in any
  web browser.


WHAT THE CODE DOES
------------------
  1. Loads the Karlan and List (2007) dataset
  2. Generates derived variables:
       - groupid: 4 political subgroups (red/blue county x red/blue state)
       - amountmat: donation amount adjusted by match ratio
  3. Drops observations with missing control variables
  4. Runs mhtexp2 with 3,000 bootstrap replications and studentized test
     statistics across the 10 specifications listed above
  5. Exports results as LaTeX and HTML tables to output/

  The output tables report: coefficients, t-statistics, single-hypothesis
  adjusted p-values (Remark 3.2), family-wise error rate corrected p-values
  (Theorem 3.1), transitivity-corrected p-values (Remark 3.8), Bonferroni
  corrections, and Holm step-down corrections.


NOTE
----
  The mhtexp2 package includes automatic calculation of treatment effects
  as well as Bonferroni and Holm p-values. For analysts who want to perform
  these corrections separately (without computing p-values via the List,
  Shaikh, and Vayalinkal method), see the package documentation at:
    https://github.com/vayalinkal/mhtexp2
