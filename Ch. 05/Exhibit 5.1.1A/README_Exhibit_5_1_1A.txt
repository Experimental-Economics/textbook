##################################################################################################
                        README — Exhibit 5.1.1A (Chapter 5)
                    Power Analysis with Varying Treatment Levels
##################################################################################################

This folder contains code to recreate Exhibit 5.1.1A from Chapter 5. The scripts simulate
power analysis for varying treatment levels (dose-response relationships) with binary outcomes.

The simulation uses a logistic model for potential outcomes:
    Y_i(d) ~ Bernoulli(p_i(d))
    p_i(d) = 1 / (1 + exp(−(−1.75 + 0.40·d + 0.05·d²)))

Treatment doses are drawn from a discrete uniform distribution over {0, 1, 2, 3, 4, 5}.
The scripts verify model recovery and simulate statistical power across different sample sizes.

Folder structure:

    code/       Scripts in Python, R, and Stata
    data/       (Empty - synthetic data generated by simulation)
    output/     Generated output files (created automatically when you run the scripts)

All three languages produce the same results. You only need to run ONE of them.
The scripts auto-detect their own location, so there is no need to manually set a
working directory or edit any file paths (except Stata - see below).


##################################################################################################
                                    Python
##################################################################################################
  File:     code/ch5_exh_5.1.1A_Stata.py

  NOTE: Despite the filename, this is the PYTHON implementation.

  Requirements: Python 3 with numpy, pandas, statsmodels, matplotlib, and tqdm.
  If missing, run in your terminal:    pip install numpy pandas statsmodels matplotlib tqdm

  How to run:
    - From a terminal:    python ch5_exh_5.1.1A_Stata.py
    - Or open the file in your IDE (VS Code, Spyder, etc.) and run it.

  Output:
    - Console output showing model summary and power analysis results
    - Optional: Graphics showing dose-response relationships (if enabled in code)


##################################################################################################
                                       R
##################################################################################################
  File:     code/ch5_exh_5.1.1A_R.R

  Requirements: R with data.table, foreach, doParallel, and boot packages.
  If missing, run in R:    install.packages(c("data.table", "foreach", "doParallel", "boot"))

  How to run:
    - In RStudio: open the file and click "Source" (or Ctrl+Shift+S / Cmd+Shift+S).
    - From a terminal:    Rscript ch5_exh_5.1.1A_R.R

  Output:
    - Console output showing model summary and power analysis results
    - Optional: Graphics showing dose-response relationships (if enabled in code)


##################################################################################################
                                     Stata
##################################################################################################
  File:     code/ch5_exh_5.1.1A_Python.do

  NOTE: Despite the filename, this is the STATA implementation.

  Requirements: Stata (no additional packages needed).

  IMPORTANT - Working Directory:
    You MUST set your working directory to the code/ folder before running this script.
    In Stata:    cd "[your path]/Ch. 5/Exhibit 5.1.1A/code"

  How to run:
    - In Stata: open the .do file and click "Do" (or select all and run).

  Output:
    - Console output showing model summary and power analysis results
    - Optional: Graphics showing dose-response relationships (if enabled in code)
